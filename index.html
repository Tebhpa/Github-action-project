1) Build & Run

Use these commands to build your image and run it on port 8080.

$ docker build -t my-nginx .
$ docker run -d --name web -p 8080:80 my-nginx
# open ‚Üí http://localhost:8080
works with your Dockerfile

2) Using a Volume

Swap content without rebuilding the image.

$ docker volume create sitevol
$ docker run -d --name web-vol -p 8080:80 -v sitevol:/var/www/html my-nginx
# now copy files into the volume:
$ docker run --rm -it -v sitevol:/data alpine sh -c 'echo "Hello from volume" > /data/index.html'
persistent

3) Bind Mount (Live Edit)

Serve a local folder instantly.

$ mkdir -p site && echo "<h1>Live Edit</h1>" > site/index.html
$ docker run -d --name web-bind -p 8080:80 -v "$PWD/site":/var/www/html my-nginx
dev-friendly

4) Health Check

Quick sanity checks while learning.

$ docker ps
$ curl -I http://localhost:8080
$ docker logs web
debug
